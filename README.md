# personal-website


## Dag 1

**Wat heb je vandaag gedaan?**
Ik heb vandaag twee curcussen gevolgd: van scroll driven animatie en view transitions.
daarna heb ik na gedacht hoe ik het project ga aanvliegen, schetsen gemaakt en een begin gemaakt aan mijn website.

**Hoe lang duurde het?**
voor de curcussen was ik zo 2 uur bezig, met de schetsen 1 uur en een begin maken aan de codes ongeveer 3 uur

**Wat heb ik geleerd?**
Ik weet nu wat er mogelijk is zou scroll driven animation kunnen begrijpen echter weet ik nog niet helemaal hoe ik dat zou moeten toepassen.

**Wat ga ik morgen doen?**
Morgen wil ik de snaren kunnen laten werken als een button en eventuele extra interacties toepassen.
